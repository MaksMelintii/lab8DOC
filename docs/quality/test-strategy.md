# Test Strategy

## Мета
Переконатися, що система відповідає функціональним та нефункціональним вимогам.

## Рівні тестування
- Unit testing
- Integration testing
- API testing
- System testing
- Security testing
- Performance testing

## Основні принципи
- кожна критична вимога повинна мати тест;
- тестування проводиться на основі SSD;
- результати тестування фіксуються.
